# Traduccion de Stonehearth
Traduccion del juego al Castellano con tonada argentina

## Estructura

- data/original → archivos originales del juego
- data/traduccion → traducción actual
- data/old → versión antigua para migración
- tools → scripts de migración y validación
- docs → normas y glosario

## Flujo

1. Migrar traducción antigua (script)
2. Traducir bloques nuevos
3. Validar con scripts
4. Revisar y mergear
